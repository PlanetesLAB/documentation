This struct contains all basic settings such as number of grid points, model radius, input and output filenames, etc. Some of these parameters always need to be set by the user, while others are optional with preset default values. There is an exception to this rule, namely when restarting citrus with previously calculated populations. In that case, none of the non-optional parameters are required.
